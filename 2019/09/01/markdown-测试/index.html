数学公式

主题使用了 KaTex 通过 $$ 标记来实现对数学公式的支持。

行间公式必须另起一行书写。

\[\alpha+\beta=\gamma\]

行内公式 \( a^2 + b^2 = c^2 \) 需要在同一行书写。

使用行内公式需要使用 Mmark 但是会使文章目录无法生成和显示。

markup = "mmark"

When \(a \ne 0\), there are two solutions to ( ax^2 + bx + c = 0 ) and they are

\[ x = {-b \pm \sqrt{b^2-4ac} \over 2a}. \]
